protocol Collection : Sequence
  • firstIndex(of:)

    Mechanica

    Returns the first index where the specified value appears in the collection. After using firstIndex(of:) to find the last position of a particular element in a collection, you can use it to access the element by subscripting.

    Note

    Same as index(of:).

    See also

  • fetchFaultedObjects()

    Mechanica

    Fetches all faulted object in one batch executing a single fetch request for all objects that we’re interested in.

    Note

    Materializing all objects in one batch is faster than triggering the fault for each object on its own.
